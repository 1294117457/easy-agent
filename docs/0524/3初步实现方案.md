   1. AgentCore - 集合层，负责：
      - 模型 API Key 管理
      - 功能插件管理
      - 个人仓库管理
      - 插件编排（工作流）
   2. 插件层 - 功能实现：
      - 数据采集插件
      - 健身房数据插件
      - 计划定制插件
      - 各插件通过 MCP (Model Context Protocol) 接口交互
   3. 用户端：
      - 管理插件
      - 配置数据来源
      - 编排插件工作流
   ```

   

## 核心设计

| 层次      | 职责                  | 举例                            |
| :-------- | :-------------------- | :------------------------------ |
| AgentCore | 集合层，统一管理      | 模型 API Key、多插件调度        |
| 插件      | 功能实现，数据处理    | 健身数据采集、地图查询、AI 分析 |
| MCP       | 插件间通信协议        | 统一的数据格式和调用规范        |
| 用户      | 配置授权 + 编排工作流 | 安装插件、拖拽串联、设置数据源  |
