Agent 个体源的核心能力是什么？
    只是一个去中心化的账户体，
    让所有人能有个proxy来介入获取现在以及未来的agent提供的便捷功能
    避免现在主流的中心化服务给用户带来的操作麻烦商业负担
插件的边界在哪里？ — 什么算"插件"，什么算"应用"？
    想要去中心化，就避免对任何平台的深入的嵌入
    所有功能都是热插拔使用，比如就是一个账户，可以通过浏览器h5页面登录使用，
    h5又有可能功能无法特别完善，或者说类似npm那样，应用上，提供一个获取热插拔功能的平台，
    比如只提供一个软件，上方主要功能是：
        1.用户登录，用户携带关键信息数据可以在任何平台上登录自己的账户
        2.获取插件的能力，agent个体源不具备任何功能，但是可以获取任何功能插件，然后用户调用使用
