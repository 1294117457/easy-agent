##### 1.预期对象

```
Workflow（工作流）    → 解决"我要让 AI 按我的步骤执行"的问题
Conversation（对话）  → 解决"我要保存多轮对话历史"的问题
Message（消息）       → 解决"对话里每条消息是什么"的问题
Prompt（模板）        → 解决"怎么让 AI 按特定角色回答"的问题
McpServer（MCP 服务器）→ 解决"AI 能调用哪些外部工具"的问题
McpTool（MCP 工具）  → 解决"每个工具是什么、怎么用"的问题
ApiKey（凭证）        → 解决"怎么让 AI 调用付费 API"的问题
```

##### 2.workflow

```
核心字段
	_nodes[],_edges[],status
方法
	node，edge的crud方法，
	hasCirle判断是否存在环
	getExecutionOrder获取执行顺序
	validate验证workflow是否合法
功能
	记录节点、执行顺序、状态，验证工作流
```

##### 3.Conversion

```
核心字段
	workflowId，_messages[]
方法
	消息的crud，validate
功能
	用户和ai对话，每个对话为一个conversation
```

##### 4.Prompt

```
核心字段

	isBuiltin的保护逻辑
功能
	用户通过设置不同prompt，给llm区分不同角色
```
