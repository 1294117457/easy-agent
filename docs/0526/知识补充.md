### electron

##### 方案对比

| 方案                     | 优点                              | 缺点                             | 适合场景                         |
| :----------------------- | :-------------------------------- | :------------------------------- | :------------------------------- |
| **Electron**             | Web 技术上手快，跨平台，生态成熟  | 包体积大（~70MB+），内存占用较高 | 需要复杂 UI 和系统能力的桌面应用 |
| **Tauri**                | 包体积小（~10MB），性能好，更安全 | Rust 后端有学习成本，生态较新    | 对性能/体积敏感的应用            |
| **NW.js**                | 更原生，可直接调用 Node           | 社区较小，文档一般               | 已有 NW.js 项目的维护            |
| **React Native Desktop** | React 语法，接近原生              | 不成熟，Windows 支持弱           | 实验性项目                       |
| **Flutter Desktop**      | 性能好，UI 精致                   | Dart 学习成本，生态不如 Web      | 全新项目，团队有 Flutter 经验    |
