## 二、问题 1：第一条消息发送失败

### 2.1 错误信息

```
错误：400 Empty input messages
```

### 2.2 问题分析

**错误链路**：

```
用户发送第一条消息
  ↓
AgentService.sendMessage() 被调用
  ↓
获取 historyWithoutCurrent = getMessages() - 1（刚添加的用户消息）
  ↓
historyWithoutCurrent.length === 0（这是第一条消息）
  ↓
recentHistory = historyWithoutCurrent.slice(-10)  // []
  ↓
构建 langchainMessages = []  // 空数组
  ↓
构建 allMessages = [SystemMessage, ...]  // 只有系统消息
  ↓
LLM 返回 400 Empty input messages
```

**核心问题**：
- 第一条消息时，`historyWithoutCurrent` 是空数组（因为 `getMessages()` 返回的数组在 append 后只有 1 条消息，slice 后是 0 条）
- 传给 LLM 的消息只有 SystemMessage，没有用户消息

### 2.3 当前代码

```typescript
// AgentService.ts
async sendMessage(conversationId, userInput, callbacks) {
  // 1. 先存储用户消息
  this.storagePort.appendMessage({ conversationId, role: 'user', content: userInput });

  // 2. 获取历史消息
  const history = this.storagePort.getMessages(conversationId);
  const historyWithoutCurrent = history.slice(0, -1);  // ⚠️ 第一条消息时为空
  const recentHistory = historyWithoutCurrent.slice(-10);

  // 3. 构建消息
  const langchainMessages = recentHistory.map(m =>
    m.role === 'user' ? new HumanMessage(m.content) : new AIMessage(m.content)
  );
  // ⚠️ 如果是第一条消息，langchainMessages 是空数组

  const allMessages = [
    new SystemMessage(systemPrompt),
    ...langchainMessages,  // ⚠️ 只有 SystemMessage
  ];

  // 4. 调用 LLM
  const result = await this.llmPort.invokeStream(allMessages, ...);
}
```

### 2.4 修复方案

**方案 A：手动添加当前用户消息（推荐）**

```typescript
async sendMessage(conversationId, userInput, callbacks) {
```

**方案 B：在获取历史消息后再添加用户消息**

```typescript
async sendMessage(conversationId, userInput, callbacks) {
  // ⚠️ 调换顺序：先获取历史，再添加用户消息
  const history = this.storagePort.getMessages(conversationId);
  const recentHistory = history.slice(-10);

  // 构建 langchainMessages
  const langchainMessages: BaseMessage[] = recentHistory.map((m) =>
    m.role === 'user'
      ? new HumanMessage(m.content)
      : new AIMessage(m.content)
  );

  // 添加当前用户消息
  langchainMessages.push(new HumanMessage(userInput));

  // 存储用户消息（持久化）
  this.storagePort.appendMessage({
    conversationId,
    role: 'user',
    content: userInput,
  });

  // 构建 allMessages
  const allMessages: BaseMessage[] = [
    ...(systemPrompt ? [new SystemMessage(systemPrompt)] : []),
    ...langchainMessages,
  ];

  // 调用 LLM
  // ...
}
```

**推荐方案 A**，原因：
1. 逻辑更清晰：先持久化，再构建消息
2. 历史消息处理逻辑不变
3. 只需在最后添加当前用户消息

2. 发送第一条消息，应该成功返回
3. 发送多条消息，检查历史上下文是否正确

## 三、问题 2：后续消息延迟

### 3.1 问题分析

可能的原因：

1. **历史消息过长**：每次都传递完整历史
2. **SystemMessage 重复**：每次都加上 SystemMessage
3. **消息数量计算错误**：导致重复添加消息

### 3.2 当前代码分析

```typescript
// 当前流程
1. appendMessage() - 存储用户消息
2. getMessages() - 获取全部历史
3. slice(-10) - 只取最近 10 条
4. 构建 langchainMessages
5. 构建 allMessages = [SystemMessage, ...langchainMessages]
```

**问题**：
- 如果 history 已经有 10 条消息，`historyWithoutCurrent.slice(-10)` 只会取 10 条
- 但实际上我们已经在步骤 1 追加了用户消息
- 所以 `getMessages()` 会返回 11 条（10 条旧 + 1 条新）
- `slice(0, -1)` 会返回 10 条旧消息
- `slice(-10)` 会返回这 10 条旧消息
- **然后我们又添加了当前用户消息**，总共 11 条

等等，让我重新分析：

```typescript
const history = this.storagePort.getMessages(conversationId);
// 如果之前有 9 条消息，现在有 10 条（9 旧 + 1 新追加的）

const historyWithoutCurrent = history.slice(0, -1);
// 返回 9 条（去掉最后一条）

const recentHistory = historyWithoutCurrent.slice(-10);
// 返回 9 条（不足 10 条）

const langchainMessages = [...recentHistory, new HumanMessage(userInput)];
// 返回 10 条（9 条历史 + 1 条当前）

const allMessages = [SystemMessage, ...langchainMessages];
// 返回 11 条
```

这样看起来是正确的...让我检查是否有其他问题。

### 3.3 可能的延迟原因

1. **重复调用 getMessages()**
2. **SystemMessage 重复添加**

### 3.4 优化方案

```typescript
async sendMessage(conversationId, userInput, callbacks) {
  console.log('[AgentService] sendMessage 开始, conversationId:', conversationId);
  console.log('[AgentService] userInput:', userInput);
  console.log('[AgentService] 当前模型:', this.llmPort.provider, this.llmPort.model);

  // 获取当前激活的 Prompt
  const activePrompt = this.storagePort.getActivePrompt();
  const systemPrompt = activePrompt?.systemPrompt || '';

  // ⚠️ 优化：先构建消息，避免重复调用 getMessages()
  // 获取历史消息（不包括当前）
  const history = this.storagePort.getMessages(conversationId);
  const historyWithoutCurrent = history.slice(0, -1);
  const recentHistory = historyWithoutCurrent.slice(-10);

  // 构建 langchain 消息
  const langchainMessages: BaseMessage[] = recentHistory.map((m) =>
    m.role === 'user'
      ? new HumanMessage(m.content)
      : new AIMessage(m.content)
  );

  // 添加当前用户消息
  langchainMessages.push(new HumanMessage(userInput));

  // 构建最终消息列表
  const allMessages: BaseMessage[] = [
    ...(systemPrompt ? [new SystemMessage(systemPrompt)] : []),
    ...langchainMessages,
  ];

  console.log('[AgentService] 发送给 LLM 的消息数:', allMessages.length);

  // ⚠️ 存储用户消息（现在才持久化）
  this.storagePort.appendMessage({
    conversationId,
    role: 'user',
    content: userInput,
  });

  try {
    let tokenCount = 0;
    const result = await this.llmPort.invokeStream(
      allMessages,
      (token) => {
        tokenCount++;
        callbacks.onToken(token);
      }
    );

    console.log('[AgentService] LLM 返回完成，总 token 数:', tokenCount);

    this.storagePort.appendMessage({
      conversationId,
      role: 'assistant',
      content: result.content,
      model: this.llmPort.model,
    });

    callbacks.onDone();
  } catch (err: unknown) {
    console.error('[AgentService] 错误:', err);
    const msg = err instanceof Error ? err.message : String(err);
    callbacks.onError(msg);
  }
}
```

**关键改动**：
1. 先获取历史消息并构建 `allMessages`
2. **最后才调用 `appendMessage()` 持久化**
3. 避免重复调用 `getMessages()`

## 四、问题 3：消息列表无滚动条

### 4.1 问题分析

当前 `MessageList.vue` 的 CSS：

```css
.message-list {
  flex: 1;
  overflow-y: auto;
  /* ... */
}
```

**问题**：
- `overflow-y: auto` 已设置，理论上应该可以滚动
- 但可能父容器 `messages-container` 的样式导致问题

当前 `ChatView.vue`：

```css
.messages-container {
  flex: 1;
  overflow: hidden;  /* ⚠️ 问题在这里 */
  padding: 0 16px;
}
```

**根本原因**：
- `.messages-container` 设置了 `overflow: hidden`
- 子元素 `.message-list` 虽然有 `overflow-y: auto`，但无法滚动
- 消息列表被限制在容器内，超出部分被隐藏

### 4.2 修复方案

**方案 A：修改 messages-container**

```css
.messages-container {
  flex: 1;
  overflow: hidden;
  display: flex;
  flex-direction: column;
  padding: 0 16px;
}

.message-list {
  flex: 1;
  overflow-y: auto;
  /* ... */
}
```

**方案 B：让 message-list 滚动（推荐）**

```css
.message-list {
  flex: 1;
  overflow-y: auto;
  min-height: 0;  /* ⚠️ 关键：允许 flex 子元素收缩 */
}

/* 同时添加滚动条样式 */
.message-list {
  scrollbar-width: thin;
  scrollbar-color: #d1d5db transparent;
}

.message-list::-webkit-scrollbar {
  width: 6px;
}

.message-list::-webkit-scrollbar-thumb {
  background: #d1d5db;
  border-radius: 3px;
}
```

## 五、综合修复代码

### 5.1 AgentService.ts 最终版本

```typescript
import { HumanMessage, AIMessage, SystemMessage } from '@langchain/core/messages';
import type { BaseMessage } from '@langchain/core/messages';
import type { ILLMPort } from '../ports/llm.port.js';
import type { IStoragePort } from '../ports/storage.port.js';

export interface SendMessageCallbacks {
  onToken: (token: string) => void;
  onDone: () => void;
  onError: (error: string) => void;
}

export class AgentService {
  constructor(
    private llmPort: ILLMPort,
    private storagePort: IStoragePort
  ) {}

  async sendMessage(
    conversationId: string,
    userInput: string,
    callbacks: SendMessageCallbacks
  ) {
    console.log('[AgentService] sendMessage 开始, conversationId:', conversationId);
    console.log('[AgentService] userInput:', userInput);
    console.log('[AgentService] 当前模型:', this.llmPort.provider, this.llmPort.model);

    // 获取当前激活的 Prompt
    const activePrompt = this.storagePort.getActivePrompt();
    const systemPrompt = activePrompt?.systemPrompt || '';

    if (!systemPrompt) {
      console.warn('[AgentService] 警告：没有配置 System Prompt');
    }

    // 获取历史消息（不包括当前刚要发送的用户消息）
    const history = this.storagePort.getMessages(conversationId);
    const historyWithoutCurrent = history.slice(0, -1);
    const recentHistory = historyWithoutCurrent.slice(-10);

    console.log('[AgentService] 历史消息数量:', historyWithoutCurrent.length);

    // 构建 langchain 消息
    const langchainMessages: BaseMessage[] = recentHistory.map((m) =>
      m.role === 'user'
        ? new HumanMessage(m.content)
        : new AIMessage(m.content)
    );

    // ⚠️ 关键：添加当前用户消息
    langchainMessages.push(new HumanMessage(userInput));

    // 构建最终消息列表
    const allMessages: BaseMessage[] = [
      ...(systemPrompt ? [new SystemMessage(systemPrompt)] : []),
      ...langchainMessages,
    ];

    console.log('[AgentService] 发送给 LLM 的消息数:', allMessages.length);
    console.log('[AgentService] 消息内容:', JSON.stringify(
      allMessages.map(m => ({
        role: m.getType(),
        content: (m.content as string).substring(0, 50)
      }))
    ));

    // ⚠️ 最后才持久化用户消息
    this.storagePort.appendMessage({
      conversationId,
      role: 'user',
      content: userInput,
    });

    try {
      let tokenCount = 0;
      const result = await this.llmPort.invokeStream(
        allMessages,
        (token) => {
          tokenCount++;
          callbacks.onToken(token);
        }
      );

      console.log('[AgentService] LLM 返回完成，总 token 数:', tokenCount);

      this.storagePort.appendMessage({
        conversationId,
        role: 'assistant',
        content: result.content,
        model: this.llmPort.model,
      });

      callbacks.onDone();
    } catch (err: unknown) {
      console.error('[AgentService] 错误:', err);
      const msg = err instanceof Error ? err.message : String(err);
      callbacks.onError(msg);
    }
  }
}
```
